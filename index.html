How To Use

Use this with ChatGPT Free manually.

1. Choose a review mode and school strategy pack.
2. Paste in any extra context or excerpts you want reviewed.
3. Click Generate Prompt.
4. Copy the prompt into ChatGPT and add your files or excerpts there if available.
